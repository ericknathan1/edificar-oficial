O Projeto

O Edificar é uma solução de software completa desenvolvida para atender às necessidades específicas das Assembleias de Deus no gerenciamento de suas Escolas Bíblicas Dominicais (EBD).

Focado na experiência do usuário e na integridade dos dados, o sistema permite que secretários e professores gerenciem matrículas, realizem chamadas e acompanhem o crescimento das turmas através de uma interface mobile intuitiva e uma API robusta.

Para o Reino

Ferramenta Open Source desenvolvida com excelência técnica para servir à comunidade cristã e demonstrar capacidade técnica de alto nível.

Ciclo de Desenvolvimento

Fase 1

Documentação & Requisitos

Levantamento de regras de negócio (Matrícula, Chamada), Diagrama ER e definição de arquitetura.

Fase 2

Prototipação (UI/UX)

Desenvolvimento dos fluxos de tela e design system utilizando Figma.

Fase 3

Backend Development

Construção da API REST com Spring Boot, implementação de segurança JWT e persistência com JPA/Hibernate.

Fase 4

Frontend Mobile

Desenvolvimento do App Android com React Native, integração com API via Axios e gerenciamento de estado.

Fase 5

DevOps & Containerização

Criação de Dockerfiles para a API e Banco de Dados, garantindo ambiente reprodutível.

Fase 6

Deploy & CI/CD

Configuração de pipelines no Jenkins e publicação dos artefatos (APK e JAR).
