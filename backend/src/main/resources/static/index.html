Como Instalar

Siga estes passos simples para instalar o Edificar App no seu Android

1

Baixe o APK

Clique no botão de download ou escaneie o QR Code para obter o arquivo APK.

2

Permita Fontes Desconhecidas

Vá em Configurações > Segurança e ative 'Permitir instalação de apps de fontes desconhecidas'.

3

Instale o Aplicativo

Abra o arquivo APK baixado e siga as instruções para concluir a instalação.

Instalação Segura

O Edificar App é seguro e livre de malware. A permissão para fontes desconhecidas é necessária apenas porque o app não está disponível na Google Play Store. Você pode desativar essa permissão após a instalação.
